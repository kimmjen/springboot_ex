# Resource 경로 지정

CSS, Javascript, Third Party Library, Images 등의 resource를 사용하기 위해서, 경로를 지정하기
'resources/static' 패키지를 생성하고, 하위에 디렉터리를 생성해서 관리

## 패키지 구조
```sh
└─src/main/resources
  ├─template
  ├─static
  |     ├─css
  |     ├─image
  |     ├─js
  |     └─webjars
  └─application.yml
```

## 소스
> WebMvcConfig.java
